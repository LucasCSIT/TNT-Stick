# TNT Stick
TNT Stick is a simple Minecraft plugin that grants the wielder of a stick some abilities related to TNT.

## Features
**Fire TNT** at far-away enemies
**Create an explosion** around you that is larger than a TNT explosion

## Controls
1. **Left Click**: Fire TNT
2. **Right Click**: Explode everything around you
